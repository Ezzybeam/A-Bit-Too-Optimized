# A Bit Too Optimized - Project Notes

A client-side Minecraft Fabric performance tune-up mod with quality presets,
a first-run setup wizard, hardware detection, and its own runtime
optimizations. Multi-version (1.21.11, 26.1.x, 26.2.x) via Stonecutter.

See `docs/superpowers/specs/` for the design and `docs/superpowers/plans/` for
implementation plans.

## Style rules (binding)

These come from the project owner and apply to all PROJECT-AUTHORED content:
Java source, JSON and other config files we write, GUI text shown to players,
log messages, and documentation we author.

- No em dashes anywhere.
- No emojis anywhere.
- Plain ASCII only.

Scope clarification: these rules apply to files we author. They do NOT apply
to vendor or tool-generated files we do not hand-write, such as the Gradle
wrapper (`gradlew`, `gradlew.bat`, `gradle/wrapper/`). Those are generated by
Gradle and may contain non-ASCII characters in their license headers and
comments. Do not hand-edit generated files to satisfy the ASCII rule; the
edits would be lost on regeneration. Reviewers should not flag non-ASCII
characters in such generated files.

## Build

- Java 21 required. The Gradle wrapper is pinned to Gradle 9.4.0. The modern
  Minecraft 26.x toolchain is Gradle 9.x + Fabric Loom 1.15 + Stonecutter 0.9.x
  (Loom 1.12+ supports Gradle 9; Stonecutter 0.9.x requires Gradle 9).
- Active version build: `./gradlew build`
- All versions: `./gradlew chiseledBuild`
- Switch active version: `./gradlew "Set active project to 26.1.2"`
- Dev client: `./gradlew runClient`
